DataTable Examples

This page shows three examples of working with DataTables:

  • All data pushed by server
  • All data pulled by client
  • Paged data queried by client (managed by server)

Example 1

Server returns all data with the page. DataTable manages paging of the whole dataset on the client side.

Example 2

Scaffolding of the table is rendered on the server side, the page is sent to a client with no data. DataTable prforms ajax call to poll the whole data-set. Paging is managed by DataTable on the client side.

Just the scafolding of the table is rendered on the server side, than the page is give to client just with a hedar and/or footer. On page load the data is polled from the server by ajax call. The paging of the hole recordset is managed by the datatable in the client side; the same as in the first example.
